ThumbnailItem
ThumbnailItem

A ThumbnailItem item is a complex item that wraps a Drawable and two Strings: a title and a subtitle. 

Class Overview

A TextItem is a very basic item that only contains a single text. The text will be displayed on a single line on screen.
